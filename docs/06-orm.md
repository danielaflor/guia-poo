# Mapeamento Objeto-Relacional

O mapeamento objeto-relacional busca aproximar o mundo das classes e objetos ao modelo relacional dos bancos de dados.

## Ideia central

- uma classe pode representar uma tabela;
- um objeto pode representar um registro;
- os atributos podem corresponder a colunas.

## Na transição Java x Python

Em Java, esse tema costuma aparecer com frameworks voltados ao ecossistema da linguagem. Em Python, o mesmo conceito continua existindo, mas com outras bibliotecas e formas de implementação.

## O que permanece

O principal não é decorar nomes de ferramentas, mas compreender a lógica do mapeamento entre objetos e dados persistidos.
