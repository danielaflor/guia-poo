# Guia de Transição de Java para Python na Programação Orientada a Objetos

Este guia foi desenvolvido para apoiar estudantes na transição entre Java e Python no contexto da Programação Orientada a Objetos. A proposta não é reiniciar o estudo dos conteúdos, mas mostrar como os conceitos já trabalhados em Java podem ser reinterpretados e aplicados em Python, preservando a base conceitual da disciplina.

## Objetivo do guia

O material busca apresentar, de forma comparativa e progressiva, como tópicos tradicionalmente abordados em Java podem ser compreendidos e implementados em Python. Mais do que destacar diferenças de sintaxe, o guia procura evidenciar a continuidade dos conceitos fundamentais da orientação a objetos.

## Como navegar

O conteúdo está organizado em páginas temáticas. Em cada seção, a proposta é retomar como determinado tópico aparecia em Java e, em seguida, mostrar como esse mesmo raciocínio pode ser desenvolvido em Python.

## Tópicos abordados

- fundamentos da Programação Orientada a Objetos;
- tratamento de exceções;
- relação entre POO e UML;
- mapeamento objeto-relacional;
- integração com banco de dados;
- geração de relatórios.
