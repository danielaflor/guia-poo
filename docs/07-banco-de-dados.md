# Banco de Dados

A integração com banco de dados permite que os dados do sistema sejam armazenados de forma persistente.

## Relação com POO

Ao desenvolver programas orientados a objetos, é comum que as classes do sistema representem entidades que também precisam ser armazenadas.

## Na transição de Java para Python

A linguagem muda, mas a necessidade de persistência permanece. O estudante deve compreender que o objetivo continua sendo registrar, recuperar, atualizar e organizar dados de forma estruturada.
