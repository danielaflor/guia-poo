# Apresentação

Este guia foi desenvolvido para apoiar estudantes na transição entre Java e Python no contexto da Programação Orientada a Objetos. A proposta não é reiniciar o estudo dos conteúdos, mas mostrar como os conceitos já trabalhados em Java podem ser reinterpretados e aplicados em Python, preservando a base conceitual da disciplina.

## Público-alvo

Este material é destinado, principalmente, a estudantes do curso técnico em informática que já tiveram contato com Programação Orientada a Objetos em Java e que passarão a desenvolver esses mesmos conteúdos em Python.

## Proposta do material

O guia foi estruturado como apoio à aprendizagem, buscando mostrar:

- o que muda de uma linguagem para outra;
- o que permanece em nível conceitual;
- quais adaptações de sintaxe e raciocínio são necessárias;
- como reutilizar a base construída anteriormente em Java.

## Organização

Cada seção do guia aborda um tópico da disciplina, retomando sua forma tradicional em Java e apresentando, em seguida, sua correspondência em Python.
