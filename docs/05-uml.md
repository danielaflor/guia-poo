# UML e POO

A UML não pertence a uma linguagem específica. Ela serve para representar a estrutura e o comportamento de sistemas orientados a objetos.

## Por que isso importa na transição

Mesmo que a implementação deixe de ser feita em Java e passe a ser feita em Python, a modelagem conceitual continua válida.

## Diagrama de classes

O diagrama de classes pode representar:

- classes;
- atributos;
- métodos;
- herança;
- associações entre objetos.

## Papel da UML neste guia

Neste material, a UML será utilizada como ponte entre o conceito e a implementação, mostrando que a lógica do sistema permanece, mesmo quando a linguagem muda.
