# Guia de Transição de Java para Python para a disciplina de Programação Orientada a Objetos.

Repositório do material de apoio à transição entre Java e Python no contexto da Programação Orientada a Objetos.

## Acesso ao guia
A documentação publicada estará disponível pelo GitHub Pages.

## Organização
O conteúdo do guia está estruturado em páginas separadas na pasta `docs`, com navegação definida no arquivo `mkdocs.yml`.
